mchain - Markov Chain Generator

mchain is a web application which provides a method for Markov chain generation based on any corpus text. Application uses a JSON-RPC protocol.

Parameters are:

  • crp - string with corpus text
  • dgr - degree. Length of generator key
  • mwr - maximal length of generated text (in words)

Sample call:

{
    "jsonrpc": "2.0",
    "method": "generate",
    "id": "12345-abcde",
    "params": [
        "Lorem ipsum dolor sit amet",
        2,
        10
    ]
}

For best results use long corpus text and degree greater than 3. Feel free to experiment with server available for you at URL: http://idun.pstmd.net:5555

Feel free to clone the repository with source code of a generator server and examples of clients written using jsonrpclib and requests modules: https://github.com/focagrande/mchain
